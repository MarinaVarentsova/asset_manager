Нужно исправить загрузчик.

1. В файле ошибок Excel не должны появляться колонки __EMPTY, __EMPTY_1, __EMPTY_2 и любые другие пустые/лишние колонки.

В итоговом файле ошибок должны быть строго 5 колонок:
- Номер документа
- ФИО эксперта
- Область производства судебной экспертизы
- Срок действия сертификата
- Ошибки

Даже если в исходном Excel есть скрытые, пустые или лишние колонки, их игнорировать.

2. Убрать проверку ФИО на количество слов.

Теперь поле "ФИО эксперта" проверяется только на заполненность, если строка непустая.
Не надо проверять, что ФИО состоит из 3 слов.

3. Исправить проверку номера документа.

Сейчас значение "№ АS 000089" падает с ошибкой.
Нужно учитывать, что в Excel пользователь может случайно ввести русскую букву "А" вместо латинской "A".

Перед проверкой номера документа нужно нормализовать значение:
- заменить русскую "А" на латинскую "A"
- заменить русскую "Р" на латинскую "P"
- заменить русскую "С" на латинскую "C"
- заменить русскую "Е" на латинскую "E"
- заменить русскую "О" на латинскую "O"
- заменить русскую "Х" на латинскую "X"
- убрать лишние пробелы
- привести множественные пробелы к одному пробелу

Допустимые форматы после нормализации:
- PS 000000
- AS 000000
- № PS 000000
- № AS 000000

В итоговый JS-файл номер документа сохранять в нормальном формате:
"№ PS 000000"
или
"№ AS 000000"

4. Добавить сортировку итоговых данных перед формированием JS-файла.

Поле:
"Срок действия сертификата"

Формат:
ДД.ММ.ГГГГ-ДД.ММ.ГГГГ

Нужно брать вторую дату — дату окончания сертификата.

Сортировка:
- сначала самые свежие сертификаты, то есть с самой поздней датой окончания;
- в конце самые старые сертификаты.

Пример:
19.04.2024-19.04.2027 должен быть выше, чем 10.01.2023-10.01.2026.

5. После исправлений проверь сценарии:
- Excel с лишними пустыми колонками не должен попадать в файл ошибок как __EMPTY.
- "№ АS 000089" должен приниматься как "№ AS 000089".
- ФИО из двух слов не должно быть ошибкой.
- Итоговый table_sert_centr_sud_expert.js должен быть отсортирован по дате окончания сертификата от новой к старой.
