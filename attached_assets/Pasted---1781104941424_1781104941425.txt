Нужно полностью переработать дизайн загрузчика реестра сертификатов.

Функциональность не менять.
Проверки не менять.
Работу с Supabase Storage не менять.
Нужно изменить только UX/UI и эмоциональное восприятие инструмента.

Цель:
Сейчас загрузчик выглядит как обычная форма.
Нужно сделать так, чтобы сотруднику было приятно пользоваться инструментом и хотелось возвращаться к нему снова.

# Общая концепция

Название концепции:

"Шахта качества реестра"

Пользователь загружает Excel-файл и вместе с персонажем ищет драгоценный камень качества текущего реестра.

Главный персонаж:

🐜 Муравей Сертификыч

Символ:

* порядка;
* аккуратности;
* проверки данных;
* трудолюбия.

Муравей не должен выглядеть детским.
Стиль — современный, аккуратный, дружелюбный.

# Стиль интерфейса

Вдохновение:

* Duolingo

* Supabase Dashboard
* современные игровые дашборды

Без мультяшности.

Основные цвета:

* темно-синий #0C2D6B

Использовать карточки.
Использовать легкие анимации.
Не перегружать экран.

# Процесс проверки

Во время проверки показывать последовательные сообщения:

⛏ Сертификыч исследует шахту...

🔍 Проверяем номера документов...

📅 Проверяем сроки действия сертификатов...

💎 Ищем ценные сертификаты...

📦 Сортируем найденные данные...

⚙ Формируем итоговый файл...

Показывать анимированный индикатор процесса.

# После проверки

Показывать большую карточку результата.

Рассчитать:

qualityPercent =
((totalRows - errorRows) / totalRows) * 100

Где:

totalRows = количество строк реестра

errorRows = количество строк, содержащих хотя бы одну ошибку

# Важно

Не добавлять:

* историю загрузок;
* БД;
* хранение статистики;
* достижения между сессиями;
* роли пользователей;
* сложные графики.

Все расчеты и виджеты должны строиться только по текущему загруженному Excel-файлу.

Загрузчик должен оставаться лёгким и быстрым.

Главная эмоция пользователя после проверки:

"Интересно, какой камень качества у меня сегодня получился?"
